# Base values

# Hit points (TP)

20 + (Constitution x 5) + (Willpower x 2)

# Life Points (LP)

60 + (Constitution x 21)

# Focus

Intelligence / 2 (can be done only once a day without stress buildup)

# Regeneration

5% of life points (LP) per day

# Stamina (EN)

(Constitution + Willpower) / 4

# Stress Resistance (SR)

(Willpower + Wisdom) / 2
